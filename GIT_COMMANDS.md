#  驻拽转 Git 驻专住

##  注 专砖 -GitHub

```bash
# 转 Git
git init

# 住驻转  拽爪
git add .

# 爪专转 commit 专砖
git commit -m "TradeMaster Pro - Initial deployment ready"

# 爪专转 branch 专砖
git branch -M main

# 住驻转 remote repository (祝 YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/trademaster-pro.git

# 注 -GitHub
git push -u origin main
```

##  注 注转

```bash
# 住驻转 砖
git add .

# 爪专转 commit
git commit -m "Update: 转专 砖"

# 注 -GitHub
git push
```

##  驻拽转 砖砖转

```bash
# 拽转 住住
git status

# 爪驻 住专
git log --oneline

# 拽转 branches
git branch

# 注 -GitHub
git pull
```
